# AI × LINE 餐飲即戰力工作坊 — 分組版

學員分組使用的單頁網站（淺色綠 + 🐢 烏龜引導），「LINE 官方餐飲產業小聚 · AI 餐飲即戰力工作坊」現場用。

- **學員頁**：`index.html`
  - 6 分頁：開始 / 工具（8 個 AI 技巧含可複製 prompt）/ 重點（缺工數據・4 成本拆解・3 客流量 ROI）/ 小考 / 提案 / 我這組
  - **章節小考**：3 章 × 3 題（共 9 題），答對 +1、公布正解與「為什麼」、每題只能作答一次
  - **計分**：登入填「你的名字」做個人計分，**個人分 1:1 累進該組總分**（`lw_team_scores`）
  - **提案 3 格**：① 活動主題（文字＋海報）② 行銷內容（多圖＋文字，前端自動壓縮）③ 獲利估算（文字為主，可附 ROI 圖）— 即時存 Supabase
- **講師展示台**：`board.html` — 各組照片牆、**各組小考總分排行**（🥇🥈🥉）、想上台置頂、隨機抽組、全螢幕投影（noindex，不對外連結）
